# FinalsPresentation
API key authentication

Group 3 - API key authentication

Project Overview
This project is a research and documentation presentation on API Key Authentication — a method for securing API endpoints by requiring clients to include a unique secret key in every request header. The server validates this key against a database and either grants access or returns a 401 Unauthorized response.
